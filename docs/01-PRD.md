## 🎯 产品定位

### 产品名称
**"简账" (Simple Ledger)** - 简单易用的记账应用

### 产品愿景
让个人和家庭轻松掌握财务状况，通过简单的记账操作实现财务自由。

### 核心价值
- **简单**：3 步完成一笔记账，学习成本接近零
- **全面**：个人 + 家庭双重账本，满足不同场景
- **智能**：自动分类、智能统计、预算提醒
- **同步**：多设备数据实时同步

### 目标用户
- 个人用户：希望记录日常收支，了解消费习惯
- 家庭用户：夫妻共同管理家庭账本，透明化家庭财务
- 小微商户：简单记录日常收支，不需要复杂财务系统

## 👥 用户画像

### 用户 A：小李（个人用户）
- **年龄**：25-35 岁
- **职业**：上班族、自由职业者
- **痛点**：
  - 月底不知道钱花哪里了
  - 想存钱但总是超支
  - 手动记账太麻烦，坚持不下去
- **需求**：
  - 快速记账
  - 清晰的统计报表
  - 预算提醒功能
  - 不需要太复杂的功能

### 用户 B：王先生（家庭用户）
- **年龄**：30-40 岁
- **职业**：公司职员
- **痛点**：
  - 夫妻各自记账，财务状况不透明
  - 不知道家庭整体消费情况
  - 需要共同管理家庭预算
- **需求**：
  - 家庭账本共享
  - 多人协作记账
  - 家庭成员消费统计
  - 预算控制

## 🎯 核心功能设计

### 1. 记账流程

```
点击"记一笔"
  ↓
选择类型（收入/支出）
  ↓
输入金额
  ↓
选择分类
  ↓
（可选）添加备注/图片
  ↓
选择日期
  ↓
确认保存
```

**交互设计原则**：
- 默认选中"支出"
- 默认今天日期
- 默认最近使用的分类
- 3 步完成基础记账（金额 → 分类 → 保存）
- 键盘自动弹出

---

### 2. 家庭账本

**权限设计**：
- **创建者**：完整权限（添加成员、删除成员、修改预算）
- **成员**：记账权限 + 查看权限
- **访客**：仅查看权限

**邀请流程**：
```
创建家庭账本
  ↓
生成邀请链接/二维码
  ↓
发送给家人
  ↓
家人点击链接加入
  ↓
开始共同记账
```

### 安全设计

#### 1. 认证与授权

- **JWT Token**：有效期 7 天，可续期
- **刷新 Token**：Refresh Token 有效期 30 天
- **权限控制**：基于角色的访问控制（RBAC）

#### 2. 数据加密

- **传输加密**：HTTPS（TLS 1.3）
- **密码加密**：bcrypt（加盐哈希）
- **敏感数据**：AES-256 加密存储（可选）

#### 3. 防护措施

- **SQL 注入防护**：使用预编译语句
- **XSS 防护**：输入验证 + 输出转义
- **CSRF 防护**：Token 验证
- **限流**：防止暴力破解和 DDoS

### Docker Compose 配置

```yaml
version: '3.8'

services:
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf
      - ./ssl:/etc/nginx/ssl
      - ./uploads:/var/www/uploads
    depends_on:
      - app
    restart: unless-stopped

  app:
    build: ./backend
    ports:
      - "8080:8080"
    environment:
      - SPRING_PROFILES_ACTIVE=production
      - MONGODB_URI=mongodb://mongodb:27017/ledger
      - REDIS_URI=redis://redis:6379
      - JWT_SECRET=${JWT_SECRET}
    depends_on:
      - mongodb
      - redis
    restart: unless-stopped

  mongodb:
    image: mongo:7.0
    ports:
      - "27017:27017"
    volumes:
      - mongodb-data:/data/db
      - ./mongo-init:/docker-entrypoint-initdb.d
    restart: unless-stopped

  redis:
    image: redis:7.2-alpine
    ports:
      - "6379:6379"
    volumes:
      - redis-data:/data
    restart: unless-stopped

volumes:
  mongodb-data:
  redis-data:
```
