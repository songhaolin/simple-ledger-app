# 记账页面 UI 设计（优化版）

> _简账 - 记账界面（根据用户反馈优化）_
> _创建者：菜菜子 🤖_

---

## 核心变化

1. **分类先行**：进入页面即显示所有分类
2. **自动键盘**：选择分类后自动弹出数字键盘
3. **独立备注**：点击备注呼出输入法
4. **支持计算**：数字键盘支持简单加减法

## 交互流程

```
进入记账页
  ↓
分类列表显示（无键盘）
  ↓
点击分类
  ↓
分类高亮
  ↓
自动弹出数字键盘
  ↓
输入金额（支持 + - × /）
  ↓
点击备注
  ↓
数字键盘收起
  ↓
呼出输入法键盘
  ↓
输入文字
  ↓
点击键盘"完成"
  ↓
保存
```
